Make Editorial Decision

Checklist

  • The Editor must add the Author as a participant using the Participants panel below.
  • The Editor must communicate with the Author if any revisions are required using the Message panel below.
  • The Editor must share the reviewers' comments with the Author using the Share Reviews button.
  • The Author must respond to any queries from the Editor in the Message panel and upload revised files using the Files panel below, if any revisions are required.
  • The Editor must record a final editorial decision using the Record Decision button on the right.
  • Add Participant
  • Attach File
  • Send Message
  • Share Reviews
  • Record Decision

Checklist

  • The Editor must attach the copyedited files to be turned into a galley using the files panel below.

  • The Layout Editor must prepare the PDF galley. This can be completed by closing this task and going to the Galleys tab.
  • The Editor must approve the PDF galley and enable it for publication. This can be completed by closing this task and going to the Galleys tab.
